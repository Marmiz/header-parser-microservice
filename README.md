# Header Parser Microservice.

## What is this?
---
A simple program that return on screen the ip, language and operative system of the visitng user.


Exeample usage:
```
{"ip":"xxx.xx.xxx.xxx","language":"en-US","software":"Macintosh; Intel Mac OS X 10_11_6"}
```
